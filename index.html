Über mich

Hallo, mein Name ist Ivan Fricker. Ich bin gerade in der Informatikmittelschule, Kantonsschule Baden als Informatiker EFZ Fachrichtung Applikationsentwicklung. Nächstes Jahr (2021) darf ich ein Praktikum machen und kann in die Berufswelt einsteigen.

In meiner Freizeit spiele ich oft Schach mit Freunden auf Chess.com. Mir macht es auch Spass, Videos und Bilder zu bearbeiten/schneiden mit Vegas Pro 15. Zurzeit programmiere ich auch in meiner Freizeit an Projekten mit HTML, CSS und JavaScript. Die einzelnen Projekte werde ich dann auf diese Webseite hochladen. Meine Webseite und die erstellten Projekte können auf meinen GitHub Profil heruntergeladen werden. https://github.com/Ivan-Fricker?tab=repositories

Projekte

1. Taschenrechner mit JavaScript

Als erstes Projekt erstellte ich einen simplen Taschenrechner mit HTML, CSS und JavaScript. HTML und CSS haben wir schon oft in der Schule genutzt. Im neusten Modul haben wir uns nun auch ein bisschen mit JavaScript beschäftigt. Ich fand JavaScript ausserordentlich spannend und fing daraufhin den Udacity JavaScript Kurs an.

Mein erstes Projekt mit JavaScript hat mir viel Spass gemacht und war eine tolle Herausforderung. Mit HTML und CSS hatte ich keine Probleme, doch bei JavaScript musste ich mich doch mehrmals mit dem Internet informieren. Auch mit meinen Kenntnissen von anderen Programmierspachen, konnte ich mich schnell an JavaScript gewöhnen.

2. Abschreibungsrechner (IDPA)

In der Schule mussten wir für die Interdisziplinäre Projektarbeit (IDPA) einen Abschreibungsrechner erstellen. Den Abschreibungsrechner haben wir mit Java und JavaFX erstellt und als Pattern haben wir MVVM benutzt. Für das Projekt haben wir auch eine fast 40 Seitige Dokumentation erstellt. Die Gruppenmitglieder waren: Emirhan Karaca, Jekathmenan Selvarajah, Niklaus Kaderli und ich. Das Programm wie auch die Dokumentation können sie auf meinen GitHub Profil herunterladen (https://github.com/Ivan-Fricker/Abschreibungsrechner-IDPA).
